# Public domain Commons image imports

We will import bird images from Wikimedia Commons using a human-in-the-loop pipeline that accepts only public-domain media for the MVP. This rejects attribution-required Commons licenses such as CC BY and CC BY-SA, even though they are freely reusable, because the app is offline-first and we want legal/ethical reuse to remain simple, auditable, and contributor-friendly.

The pipeline will use scientific names as species identity, query Commons MediaSearch, lazily validate each candidate's Commons metadata against a conservative public-domain whitelist, open the Commons file page for human source review, download a thumbnail-sized source when available, produce a WebP under 100KB with a 960px long-edge cap, then ask for human approval of the processed image before storing it. A bulk import run will read an explicit species JSON file, skip species that already have one MVP image, remember rejected candidates outside `public/`, continue across species failures, record birds with no approved image in `data/missing-birds.json`, update the bundled bird catalog after successful imports, and ship image provenance JSON next to each processed image.
