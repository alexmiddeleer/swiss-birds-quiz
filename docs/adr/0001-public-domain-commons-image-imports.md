# Open-license Commons image imports

We will import bird images from Wikimedia Commons using a human-in-the-loop pipeline that accepts public-domain media and open licenses we can visibly honor in the app, including CC BY and CC BY-SA. This keeps the app legal and ethical while widening coverage for common Swiss birds whose usable Commons photos are often attribution-required.

The pipeline will use scientific names as species identity, query Wikimedia Commons file search through the Action API, lazily validate each candidate's Commons metadata against a conservative reusable-license whitelist, open the Commons file page for human source review, download a thumbnail-sized source when available, produce a WebP under 100KB with a 960px long-edge cap, then ask for human approval of the processed image before storing it. A bulk import run will read an explicit species JSON file, skip species that already have one MVP image, remember rejected candidates outside `public/`, continue across species failures, record birds with no approved image in `data/missing-birds.json`, update the bundled bird catalog after successful imports, and ship image provenance JSON next to each processed image so the app can show visible attribution.
